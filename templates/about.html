About

This is a web application that aims to enrich your Hearthstone experience with functionality such as deck building, sharing, and tracking, as well as provide a collaboration and viewing platform for those decks. You MUST be logged in via your google account to use the majority of features.



This website is NOT optimized for mobile. We have plans to support mobile but this will come much later down the line. This application was also developed for Google Chrome, so other browsers might not be supported. Please bear with us as we correct these issues over time.



If you'd like to contribute to this project, you can fork it over on GitHub.
